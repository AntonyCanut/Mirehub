# Raccourcis clavier

## Terminal

| Raccourci | Action |
|-----------|--------|
| `Cmd+T` | Nouvel onglet terminal |
| `Cmd+W` | Fermer le pane actif (ou l'onglet s'il n'a qu'un pane) |
| `Cmd+D` | Split horizontal (ajoute un pane a droite) |
| `Cmd+Shift+D` | Split vertical (ajoute un pane en bas) |
| `Cmd+Shift+Enter` | Zoom/dezoom le pane actif (plein ecran) |
| `Cmd+Alt+←` | Naviguer vers le pane de gauche |
| `Cmd+Alt+→` | Naviguer vers le pane de droite |
| `Cmd+Alt+↑` | Naviguer vers le pane du haut |
| `Cmd+Alt+↓` | Naviguer vers le pane du bas |
| `Cmd+Shift+[` | Onglet precedent (scope workspace) |
| `Cmd+Shift+]` | Onglet suivant (scope workspace) |
| `Cmd+1` a `Cmd+9` | Activer l'onglet N du workspace |

## Workspaces

| Raccourci | Action |
|-----------|--------|
| `Cmd+Shift+N` | Nouveau workspace |
| `Cmd+Shift+[` | Workspace precedent (dans la sidebar) |
| `Cmd+Shift+]` | Workspace suivant (dans la sidebar) |

## Editeur de fichiers (Monaco)

| Raccourci | Action |
|-----------|--------|
| `Cmd+S` | Sauvegarder le fichier |

## Git

| Raccourci | Action |
|-----------|--------|
| `Cmd+Enter` | Creer un commit (dans la zone de message de commit) |

## Fichiers (SidebarFileTree)

| Raccourci | Action |
|-----------|--------|
| `Cmd+Clic` | Selectionner un fichier pour comparaison diff (2 fichiers max, auto-ouverture du diff) |

## Modales

| Raccourci | Action |
|-----------|--------|
| `Escape` | Fermer la modale / annuler |
| `Enter` | Confirmer l'action |

## DevTools (developpement)

| Raccourci | Action |
|-----------|--------|
| `Cmd+Alt+I` | Ouvrir/fermer les DevTools Electron |
